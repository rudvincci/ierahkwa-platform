## Overview

Mamey.OpenBanking provides Open Banking API specifications and models for the Mamey framework. It includes account information, payment initiation, and transaction models following Open Banking standards.

### Key Features

- **Open Banking Specifications**: OpenAPI-based specifications
- **Account Information**: Account access consent and balance models
- **Payment Models**: Payment initiation models
- **Transaction Models**: Transaction processing models
