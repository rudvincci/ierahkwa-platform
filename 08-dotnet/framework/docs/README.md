# Mamey Framework Documentation

**Welcome to the comprehensive documentation for the Mamey Framework**

The Mamey Framework is a modular, enterprise-grade .NET microservices framework consisting of **110+ independent libraries** that work together to address common infrastructural challenges in building scalable, resilient, and maintainable microservices.
