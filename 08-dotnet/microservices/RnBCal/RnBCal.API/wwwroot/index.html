API Endpoints

POST /api/calendar/generate-ics

Generate ICS calendar file for a booking

POST /api/calendar/calendar-links

Get direct calendar links for all providers

POST /api/calendar/sync

Full calendar sync with ICS and links generation

POST /api/calendar/send-confirmation

Send booking confirmation email with calendar attachment

GET /api/calendar/google/oauth-url

Get Google Calendar OAuth authorization URL

POST /api/bookings

Create a new booking

GET /api/bookings

Get all bookings

GET /api/bookings/{id}

Get specific booking by ID

GET /api/bookings/stats

Get booking statistics and analytics
