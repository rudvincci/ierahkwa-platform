Inventory Manager Pro - User Guide

Getting Started

First Login

Use the default credentials to log in:

  • Username: admin
  • Password: admin123
Security: Change your password immediately after first login via File > Change Password.

Main Interface

Navigation

  • Ribbon Menu: Main toolbar at the top with all features organized by tabs
  • Navigation Pane: Left sidebar for quick access to main sections
  • Status Bar: Bottom bar showing user, database status, and stock summary

Product Management

Adding Products

  1. Go to Inventory > Products > New Product
  2. Fill in the required fields:
    • Code (auto-generated)
    • Name
    • Purchase Price
    • Sale Price
  3. Set optional fields (Category, Supplier, Barcode, etc.)
  4. Click Save
Tip: The product code is auto-generated but can be customized.

Stock Operations

Stock In (Purchase)

  1. Go to Inventory > Stock In
  2. Search and select a product
  3. Enter quantity and unit price
  4. Add to list
  5. Repeat for more products
  6. Click Save Movement

Stock Out (Sale)

  1. Go to Inventory > Stock Out
  2. Follow same process as Stock In
  3. System validates available stock

Quick Stock Entry

Use Quick Stock Entry for barcode scanner support:

  • Connect barcode scanner
  • Select movement type (In/Out)
  • Scan products - they're automatically added
  • Edit quantities if needed
  • Save all at once

Search Features

Quick Search

Press F3 anywhere to open Quick Search:

  • Search by code, barcode, or name
  • Results appear instantly
  • Press Enter to view details

Grid Search

Each grid has a built-in search bar for filtering displayed data.

Reports & Exports

Available Reports

Report Description

Export Formats

  • Excel: Full data with formatting
  • PDF: Print-ready documents
  • HTML: Web viewable
  • TXT/CSV: Plain text

Backup & Restore

Creating a Backup

  1. Go to Tools > Backup Database
  2. Choose save location
  3. Click Save

Restoring from Backup

  1. Go to Tools > Restore Database
  2. Select backup file
  3. Confirm restore
  4. Application will restart
Warning: Restoring will replace ALL current data!
