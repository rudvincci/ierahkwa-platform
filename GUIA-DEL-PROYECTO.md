## Guia Completa del Proyecto — Plataforma Digital Soberana

**Version:** 2.0 | **Fecha:** 22 Febrero 2026 | **FutureHead Group**

---

## QUE ES ESTO

Este es el sistema digital completo de la primera nacion digital indigena soberana del mundo. Contiene **296 plataformas** que reemplazan toda la infraestructura tecnologica de Big Tech (Google, Facebook, Amazon, etc.) con alternativas 100% controladas por pueblos indigenas.

**Alcance:** 72 millones de indigenas en las Americas. 476 millones a nivel mundial.

## COMO EMPEZAR

### Para ver las plataformas:
```bash
# Abre cualquier plataforma en tu navegador
open 02-plataformas-html/correo-soberano/index.html
open 02-plataformas-html/voz-soberana/index.html
open 06-dashboards/mamey-dashboard.html
```

### Para entender el proyecto:
1. Lee `01-documentos/inversores/WHITEPAPER-IERAHKWA.md`
2. Lee `01-documentos/auditoria/MAPA-COMPLETO-PLATAFORMA-SOBERANA.md`
3. Lee `01-documentos/legal/GOVERNANCE-CONSTITUTION.md`
4. Lee `ARCHITECTURE.md` para la arquitectura del sistema

### Para desarrolladores:
```bash
# 1. Setup automatico
./setup.sh

# 2. Comandos rapidos
make start          # Levantar todo
make status         # Verificar estado
make build-all      # Compilar todo
make test           # Correr tests
make help           # Ver todos los comandos

# 3. Leer documentacion tecnica
cat 01-documentos/tecnico/DEVELOPER-ONBOARDING.md
cat 05-api/API-SPECIFICATION.md
cat CONTRIBUTING.md
```
