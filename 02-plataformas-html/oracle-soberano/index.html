Módulos de la Plataforma

10 herramientas soberanas de grado empresarial

💲

Price Feeds WAMPUM Multi-Asset

Feeds de precios en tiempo real para WAMPUM y 200+ activos. Aggregation multi-source con detección de outliers.

🌦️

Datos Climáticos para Agricultura

Datos meteorológicos verificados para smart contracts agrícolas. Temperatura, lluvia, humedad y predicciones para 574 territorios.

🏥

Datos de Salud Pública

Indicadores de salud pública anonimizados para protocolos de seguros y planificación. Epidemiología, vacunación, nutrición.

🎲

VRF Randomness Verificable

Generación de números aleatorios verificables on-chain. Para loterías, gaming, selección de jurados y asignación justa.

🖥️

Computación Off-Chain Segura

Ejecución de computación pesada off-chain con pruebas verificables on-chain. Machine learning, simulaciones, analytics.

🌍

Nodos Soberanos por Nación

Cada nación opera nodos oracle con stake propio. Descentralización real, no corporativa.

📊

Aggregation Multi-Source

Agregación de múltiples fuentes de datos con mediana ponderada. Resistente a manipulación y outliers.

💓

Heartbeat Monitoring

Monitoreo de latido de nodos con alertas automáticas. Detección de nodos caídos y failover instantáneo.

⭐

Data Quality Scoring

Puntuación de calidad de datos por fuente. Incentivos para datos precisos, penalizaciones para inexactos.

⚖️

Dispute Resolution DAO

Resolución de disputas sobre datos mediante votación DAO. Arbitraje descentralizado para feeds controversiales.

🔌 API Endpoints

REST + gRPC + WebSocket para integraciones
GET/api/v1/oracle/prices/{asset}

Precio actual del activo con timestamp, sources y confidence.

GET/api/v1/oracle/weather/{territory}

Datos climáticos del territorio con histórico y predicción.

GET/api/v1/oracle/health/{region}

Indicadores de salud pública de la región.

POST/api/v1/oracle/vrf

Solicitar número aleatorio verificable. Params: seed, callback.

GET/api/v1/oracle/feeds

Lista de todos los feeds disponibles con frecuencia y quality score.

GET/api/v1/oracle/nodes

Nodos oracle activos con stake, uptime y performance score.
