🔌 API Endpoints

REST + gRPC + WebSocket para integraciones
POST/api/v1/defi/lend

Depositar activos en pool de lending. Params: asset, amount, duration.

POST/api/v1/defi/borrow

Solicitar préstamo. Params: collateral, borrow_asset, amount, ltv.

GET/api/v1/defi/pools

Lista de pools activos con TVL, APY y métricas de riesgo.

GET/api/v1/defi/yields/{protocol}

Rendimientos históricos y proyectados por protocolo.

POST/api/v1/defi/swap

Intercambio de tokens con routing óptimo multi-pool.

GET/api/v1/defi/tvl

Total Value Locked global y desglose por protocolo y nación.
