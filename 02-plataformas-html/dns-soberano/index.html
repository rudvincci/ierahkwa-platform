Módulos de la Plataforma

10 herramientas soberanas de grado empresarial

🔒

DNS-over-HTTPS/TLS/QUIC

Resolución DNS completamente cifrada con soporte DoH, DoT y DoQ. Imposible para ISPs o gobiernos interceptar o manipular tus consultas DNS.

🚫

Bloqueo de Ads y Trackers

Bloqueo a nivel de red de anuncios, trackers, malware y telemetría. Listas curadas para proteger a toda la comunidad sin instalar extensiones.

🌐

Dominio .nation Nativo

Sistema de dominios .nation soberano, independiente de ICANN. Registro, gestión y resolución de dominios bajo gobernanza indígena.

🛡️

DNSSEC Integrado

Firmas criptográficas en cada respuesta DNS para prevenir spoofing y envenenamiento de cache. Validación automática end-to-end.

📡

Cache Distribuido 847 Nodos

Cache DNS distribuido en 847 nodos soberanos mundiales. Resolución ultra-rápida con redundancia geográfica y failover automático.

👁️

Zero Logs Privacy

Política estricta de cero logs: ninguna consulta DNS se almacena, registra o analiza. Privacidad verificable con auditorías públicas.

⚛️

Anti-Spoofing Post-Cuántico

Protección contra ataques DNS con algoritmos post-cuánticos Kyber-768. Futureproof contra computadoras cuánticas.

📋

Listas de Bloqueo Personalizables

Listas de bloqueo configurables por comunidad: ads, trackers, adult content, gambling. Import/export compatible con Pi-hole y AdGuard.

📊

Dashboard de Estadísticas

Panel con estadísticas en tiempo real: consultas/segundo, dominios bloqueados, latencia promedio, distribución geográfica, top domains.

🔌

API para Integración

API REST completa para gestionar configuración DNS, listas de bloqueo, dominios .nation y estadísticas desde cualquier aplicación.

🔌 API Endpoints

REST + gRPC + WebSocket para integraciones
GET/api/v1/dns/resolve

Resolver dominio. Params: domain, type (A, AAAA, MX, TXT, CNAME).

POST/api/v1/dns/blocklist

Gestionar listas de bloqueo. Params: action (add/remove), list_url.

GET/api/v1/dns/stats

Estadísticas en tiempo real: queries/s, blocked, latency, top domains.

POST/api/v1/dns/domain

Registrar/gestionar dominio .nation. Params: domain, records, ttl.

GET/api/v1/dns/cache

Estado del cache DNS: hit rate, tamaño, distribución, TTLs activos.

POST/api/v1/dns/config

Actualizar configuración DNS. Params: upstream, doh_enabled, blocklists.
