## 5. API Endpoints

### 5.1 CaseService (:9500)

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/case/create` | Create new case (encrypted) |

| PUT | `/api/case/{id}/status` | Update case status |
| POST | `/api/case/{id}/note` | Add case note (encrypted) |
| GET | `/api/case/worker/{id}/caseload` | Get caseworker's active cases |
| POST | `/api/case/{id}/seal` | Seal case (witness protection) |
| DELETE | `/api/case/{id}` | Right-to-deletion (permanent) |

|--------|----------|-------------|
