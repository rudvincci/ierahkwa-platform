# NAE Turismo Inteligente — Guna Yala Smart Tourism

> Plataforma de gestión turística inteligente para las 49 comunidades y 365 islas de Guna Yala, Panamá

## Overview

NAE es la plataforma de turismo inteligente diseñada específicamente para Guna Yala — el territorio indígena más visitado de Panamá. Gestiona el flujo turístico completo: reservas, transporte 4x4, lanchas, cabañas, excursiones, con dashboards por rol, calendario de disponibilidad en tiempo real, QR soberano de rastreo, control de capacidad por isla y pagos WAMPUM. Es el modelo pionero que inspira a Turismo Soberano (574 territorios) y Eco-Turismo de las Américas.

## El Problema de Guna Yala

- **85,000+ turistas/año** llegan sin control ni coordinación
- Tour operadores de Ciudad de Panamá controlan el flujo y se quedan con las ganancias
- No hay sistema de capacidad — islas populares reciben más turistas de lo sostenible
- Transportistas, lancheros y cabañeros no tienen visibilidad de reservas
- Pagos en efectivo sin transparencia ni distribución justa
- El Congreso General Guna no tiene datos en tiempo real para tomar decisiones

## Solución NAE

### 6 Zonas Turísticas

| Zona | Islas Principales | Capacidad/Día |
|------|------------------|---------------|
| 1. Carti / El Porvenir | Nalunega, Wichub-Wala, Isla Perro | 300 |
| 2. Río Sidra / Nusagandi | Naranjo Chico, Kuanidup | 200 |
| 3. Playón Chico | Centro cultural, talleres molas | 150 |
| 4. Achutupu / Mamitupu | Islas vírgenes, camping | 80 |
| 5. Ailitupu / Isla Tigre | Selva, medicina, arqueología | 100 |
| 6. Ustupu / Ogobsucun | Capital, Congreso Guna | 500 |

### 5 Dashboards por Rol

1. 🚐 **Transportista** — 18 activos. 4x4 Panamá→Carti, horarios, reservas, GPS
2. 🚤 **Lanchero** — 45 activos. Calendario, capacidad, clima, manifiestos
3. 🏠 **Dueño de Cabaña** — 120 cabañas. Ocupación, precios, limpieza, reseñas
4. 🎒 **Tour Operador** — 15 operadores. Paquetes, coordinación, facturación
5. 🛡️ **Administrador** — Congreso + zonales. Control total, capacidad, reportes

### Flujo QR (8 Pasos)

Reserva → QR viaje → 4x4 scan → Lancha scan → Cabaña check-in → Excursiones → Monitoreo → Check-out

### Distribución de Ingresos WAMPUM

- Cabaña: 40% | Lancha: 20% | Transporte: 15%
- Tour Operador: 10% | Congreso Guna: 10% | Plataforma: 5%

## Tecnología

- HTML5 + CSS3 + JavaScript vanilla (zero dependencias)
- PostGIS para mapas de islas y zonas offline
- P2P Mesh Sync para islas sin señal
- Cifrado post-cuántico Kyber-768
- MameyNode blockchain para pagos
- 7 AI Agents de seguridad
- PWA offline-first (funciona en islas sin internet)
