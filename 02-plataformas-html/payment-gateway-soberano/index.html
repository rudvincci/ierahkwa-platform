Módulos de la Plataforma

10 herramientas soberanas de grado empresarial

🛒

Checkout Embebido Universal

Widget de checkout que se embebe en cualquier sitio web con 3 líneas de código. Detección automática de wallet, selección de moneda y confirmación instantánea.

🔄

Conversión Automática WAMPUM

Conversión instantánea de cualquier criptomoneda recibida a WAMPUM o stablecoin soberana. El comercio recibe siempre el monto exacto sin riesgo de volatilidad.

🏪

POS para Tiendas Físicas

Aplicación POS que funciona en tablets y smartphones. Genera QR dinámicos, acepta NFC y muestra confirmación visual y sonora al completar pago.

📄

Facturación Recurrente Cripto

Subscripciones y pagos recurrentes con smart contracts. El cliente autoriza una vez y los cobros se ejecutan automáticamente según frecuencia.

📱

QR Code Payments

Generación de QR codes dinámicos con monto, moneda y metadata embebidos. Escaneo con cualquier wallet compatible con estándares soberanos.

🔌

API REST para E-Commerce

API RESTful completa para integración con WooCommerce, Shopify, Magento y plataformas custom. SDKs en 8 lenguajes de programación.

🔔

Webhook Notificaciones

Webhooks configurables para eventos de pago: creado, confirmado, completado, expirado, reembolsado. Retry automático con backoff exponencial.

💱

Multi-Currency Settlement

Liquidación en la moneda que elija el comercio: WAMPUM, stablecoin, BTC o fiat equivalente. Frecuencia de liquidación configurable.

↩️

Refund Management

Reembolsos parciales o totales con un click. Smart contract ejecuta devolución a la wallet original del comprador automáticamente.

📊

Dashboard de Ventas

Panel de control con métricas de ventas, conversión, monedas más usadas, clientes recurrentes y gráficos de tendencia.

🔌 API Endpoints

REST + gRPC + WebSocket para integraciones
POST/api/v1/gateway/charge

Crear cargo de pago. Params: amount, currency, description, redirect_url.

GET/api/v1/gateway/status

Estado de un pago: pending, confirmed, completed, expired, refunded.

POST/api/v1/gateway/invoice

Crear factura con items, impuestos y descuentos. Genera link de pago.

GET/api/v1/gateway/transactions

Historial de transacciones con filtros por fecha, moneda y estado.

POST/api/v1/gateway/refund

Reembolsar pago completo o parcial. Params: charge_id, amount.

POST/api/v1/gateway/webhook

Configurar webhook endpoint. Params: url, events, secret.
