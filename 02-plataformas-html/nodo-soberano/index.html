Módulos de la Plataforma

10 herramientas soberanas de grado empresarial

🖥️

Nodos RPC Dedicados

Endpoints RPC dedicados con baja latencia y alta disponibilidad. Soporte para eth_call, eth_sendTransaction y todos los métodos estándar de MameyNode.

🔌

WebSocket Subscriptions

Conexiones WebSocket persistentes para subscripciones en tiempo real: nuevos bloques, logs de contratos, transacciones pendientes y cambios de estado.

📚

Archive Nodes Históricos

Nodos archive que almacenan el estado completo de cada bloque desde el genesis. Queries históricos sin límite temporal para auditoría y análisis.

📱

Light Nodes para Mobile

Nodos ligeros optimizados para dispositivos móviles. Verificación de transacciones sin descargar toda la blockchain, usando pruebas Merkle.

⚖️

Load Balancer Inteligente

Distribución inteligente de requests entre 847 nodos considerando latencia, carga y geolocalización. Failover automático sin interrupción.

🌍

Multi-Region 19 Naciones

Nodos distribuidos en las 19 naciones del ecosistema. Routing geográfico para mínima latencia y máxima resiliencia ante desconexiones regionales.

🔑

API Key Management

Sistema de API keys con permisos granulares, rate limits configurables y métricas de uso por key. Rotación automática y revocación instantánea.

🚦

Rate Limiting Justo

Rate limiting basado en necesidad, no en capacidad de pago. Cuotas generosas para DApps comunitarias y límites proporcionales al stake WAMPUM.

📟

Health Monitoring

Monitoreo continuo de salud de cada nodo: latencia, sync status, peer count, disk usage. Alertas automáticas y auto-healing.

📊

Dashboard de Infraestructura

Panel de control con métricas de toda la red: nodos activos, requests/segundo, latencia promedio, distribución geográfica y uptime.

🔌 API Endpoints

REST + gRPC + WebSocket para integraciones
POST/api/v1/node/rpc

Ejecutar llamada RPC a MameyNode. Params: method, params, block.

GET/api/v1/node/status

Estado de la red de nodos: activos, sync, latencia, throughput.

POST/api/v1/node/subscribe

Crear subscripción WebSocket. Params: event_type, filters.

GET/api/v1/node/archive

Query a archive node. Params: method, params, block_number.

GET/api/v1/node/health

Health check de nodo específico o de toda la red.

POST/api/v1/node/apikey

Crear API key con permisos y rate limits. Params: name, permissions.
