🔌 API — WAMPUM Exchange

REST + WebSocket sobre MameyNode blockchain (Chain ID 574)
POST/api/v1/exchange/order

Crear orden (limit/market/stop). Params: pair, side, price, amount, type.

GET/api/v1/exchange/orderbook/{pair}

Libro de ordenes con profundidad. Params: depth (default 20).

GET/api/v1/exchange/ticker/{pair}

Precio actual, volumen 24h, high/low, cambio %.

GET/api/v1/exchange/trades/{pair}

Trades recientes. Params: limit (default 50).

GET/api/v1/exchange/klines/{pair}

Candlestick data. Params: interval (1m/5m/1h/1d), limit.

POST/api/v1/exchange/cancel/{orderId}

Cancelar orden abierta.

GET/api/v1/exchange/wallet/balances

Balances de todas las monedas (WPM, IGT-*, SNT-*).

POST/api/v1/exchange/wallet/transfer

Transferir entre wallets soberanos. Settled en MameyNode.

GET/api/v1/exchange/snt/list

Lista de los 574 SNT tokens con precio, volumen y status.

WS/ws/exchange/stream

WebSocket: orderbook updates, trades, ticker, klines en tiempo real.
