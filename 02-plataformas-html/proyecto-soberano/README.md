# ProyectoSoberano — Sovereign Project Management | MameyNode

> Plataforma soberana de Proyecto para la infraestructura digital de 574 naciones tribales - Ierahkwa Ne Kanienke

## Resumen

**ProyectoSoberano** es una plataforma del ecosistema **Ierahkwa Ne Kanienke**, parte de **NEXUS Forja (Desarrollo & DevOps)**. Diseñada para la soberanía digital de 72 millones de personas indígenas en 19 naciones y 574 tribus.

1. **📋 Kanban Boards Visual task management with customizable columns, WIP limits, swimlanes, and drag-and-drop cards.**
2. **🔗 API Integrations Connect with CodeSoberano, CorreoSoberano, and 30+ sovereign platforms through the MameyNode API.**
3. **Why Sovereign Project Management?**
4. **🛡️ Own Your Roadmap Corporate PM tools mine your data to understand market trends. ProyectoSoberano keeps your strategy **
5. **💰 Budget Tracking Real-time project budgets with WAMPUM token integration. Track costs, revenue forecasts, and ROI.**
6. **Everything Your Team Needs From agile sprints to long-term roadmaps, manage every aspect of your projects with sovereign**
7. **Everything Your Team Needs**
8. **📖 Project Wikis Collaborative documentation with rich text, code blocks, diagrams, and version history built in.**
9. **🔐 Encrypted Collaboration Every board, task, comment, and attachment is encrypted end-to-end. Not even platform admins c**
10. **📈 Gantt Charts Interactive timeline views with dependencies, milestones, critical path analysis, and baseline tracking.**

## Tecnologías

- **Frontend**: HTML5 semántico, CSS3, JavaScript vanilla
- **Design System**: `ierahkwa.css` (shared)
- **Seguridad**: `ierahkwa-security.js` — encriptación post-quantum Kyber-768
- **AI**: `ierahkwa-agents.js` — 7 agentes autónomos anti-fraude
- **Protocolos**: `ierahkwa-protocols.js` — comunicación soberana
- **Red**: `ierahkwa-interconnect.js` — interconexión entre plataformas
- **PWA**: Service Worker + manifest.json — funciona offline

## Instalación

```bash
# Clonar el repositorio
git clone https://github.com/rudvincci/ierahkwa-platform.git

# Navegar a la plataforma
cd 02-plataformas-html/proyecto-soberano

# Abrir en navegador (no requiere servidor)
open index.html
```
