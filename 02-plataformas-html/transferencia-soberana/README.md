# TransferenciaSoberana — Sovereign File Transfer | MameyNode

> Plataforma soberana de Transferencia para la infraestructura digital de 574 naciones tribales - Ierahkwa Ne Kanienke

## Resumen

**TRANSFERENCIA SOBERANA** es una plataforma del ecosistema **Ierahkwa Ne Kanienke**, parte de **NEXUS Forja (Desarrollo & DevOps)**. Diseñada para la soberanía digital de 72 millones de personas indígenas en 19 naciones y 574 tribus.

## Características Principales

1. **POWERFUL TRANSFER CAPABILITIES Everything you need for secure, fast, and reliable file transfers across the sovereign ne**
2. **🛡️ Digital Sovereignty Your files never touch Big Tech servers. Every transfer stays on sovereign infrastructure control**
3. **📊 Bandwidth Control Fine-grained speed limits per transfer, per connection. Prioritize critical files automatically.**
4. **SUPPORTED PROTOCOLS**
5. **WHY SOVEREIGN FILE TRANSFER?**
6. **🔒 Encrypted Tunnels Post-quantum E2E encryption on every transfer. AES-256 + Kyber1024 hybrid cryptography built in.**
7. **🔄 Directory Synchronization Two-way directory sync with conflict resolution. Keep local and remote files perfectly align**
8. **☁️ S3-Compatible Storage Connect to any S3-compatible object storage including sovereign cloud and MameyNode decentraliz**
9. **🔐 Zero-Knowledge Architecture We cannot see your files. Post-quantum encryption ensures that only the sender and receive**
10. **📤 Drag & Drop Transfers Intuitive drag-and-drop interface with real-time progress indicators and transfer queue manageme**

## Tecnologías

- **Frontend**: HTML5 semántico, CSS3, JavaScript vanilla
- **Design System**: `ierahkwa.css` (shared)
- **Seguridad**: `ierahkwa-security.js` — encriptación post-quantum Kyber-768
- **AI**: `ierahkwa-agents.js` — 7 agentes autónomos anti-fraude
- **Protocolos**: `ierahkwa-protocols.js` — comunicación soberana
- **Red**: `ierahkwa-interconnect.js` — interconexión entre plataformas
- **PWA**: Service Worker + manifest.json — funciona offline

## Instalación
