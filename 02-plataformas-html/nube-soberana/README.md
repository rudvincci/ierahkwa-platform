# Nube Soberana — Sovereign Cloud & Collaboration Platform | Ierahkwa

> Plataforma soberana de Nube para la infraestructura digital de 574 naciones tribales - Ierahkwa Ne Kanienke

## Resumen

**NUBE SOBERANA** es una plataforma del ecosistema **Ierahkwa Ne Kanienke**, parte de **NEXUS Forja (Desarrollo & DevOps)**. Diseñada para la soberanía digital de 72 millones de personas indígenas en 19 naciones y 574 tribus.

## Tecnologías

- **Frontend**: HTML5 semántico, CSS3, JavaScript vanilla
- **Design System**: `ierahkwa.css` (shared)
- **Seguridad**: `ierahkwa-security.js` — encriptación post-quantum Kyber-768
- **AI**: `ierahkwa-agents.js` — 7 agentes autónomos anti-fraude
- **Protocolos**: `ierahkwa-protocols.js` — comunicación soberana
- **Red**: `ierahkwa-interconnect.js` — interconexión entre plataformas
- **PWA**: Service Worker + manifest.json — funciona offline

## Instalación

# Navegar a la plataforma
