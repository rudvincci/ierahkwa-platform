# Call Center Soberano — Centro de Contacto Omnicanal para Naciones Soberanas

> Centro de contacto omnicanal que reemplaza Asterisk, FreePBX y VICIdial. PBX soberano con VoIP SIP/WebRTC, IVR inteligente con IA en 14 idiomas indígenas, graba

## Resumen

**Call Center Soberano** es una plataforma del ecosistema **Ierahkwa Ne Kanienke**, parte de **NEXUS Forja (Desarrollo & DevOps)**. Diseñada para la soberanía digital de 72 millones de personas indígenas en 19 naciones y 574 tribus.

## Tecnologías

- **Frontend**: HTML5 semántico, CSS3, JavaScript vanilla
- **Design System**: `ierahkwa.css` (shared)
- **Seguridad**: `ierahkwa-security.js` — encriptación post-quantum Kyber-768
- **AI**: `ierahkwa-agents.js` — 7 agentes autónomos anti-fraude
- **Protocolos**: `ierahkwa-protocols.js` — comunicación soberana
- **Red**: `ierahkwa-interconnect.js` — interconexión entre plataformas
- **PWA**: Service Worker + manifest.json — funciona offline

## Instalación

```bash
# Clonar el repositorio
git clone https://github.com/rudvincci/ierahkwa-platform.git

# Navegar a la plataforma
cd 02-plataformas-html/callcenter-soberano

# Abrir en navegador (no requiere servidor)
open index.html
```
