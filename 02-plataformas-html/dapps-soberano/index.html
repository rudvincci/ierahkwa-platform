🔌 API Endpoints

REST + gRPC + WebSocket para integraciones
POST/api/v1/dapps/create

Crear nuevo proyecto DApp. Params: name, template, chain, framework.

POST/api/v1/dapps/deploy

Desplegar DApp. Params: project_id, chain, env, constructor_args.

GET/api/v1/dapps/templates

Lista de templates disponibles con descripción y categoría.

POST/api/v1/dapps/test

Ejecutar tests del proyecto. Params: project_id, test_suite, coverage.

GET/api/v1/dapps/status/{project_id}

Estado del proyecto: tests, deploys, contratos activos.

POST/api/v1/dapps/interact

Interactuar con contrato. Params: address, function, args, chain.
