Modulos de la Plataforma

10 herramientas soberanas contra la desinformacion

🔍

Verificacion de Origen

Tracking de metadata blockchain para fuentes de noticias. Cada pieza de informacion lleva un hash de origen inmutable con timestamp, geolocation y cadena editorial completa.

⛓

Cadena de Custodia

Cadena inmutable que muestra quien financio, edito y compartio cada pieza de informacion. Transparencia total del flujo informativo desde la fuente original hasta el consumidor final.

🔐

ZK-Whistleblower

Zero-knowledge proofs para informantes anonimos. Permite verificar la legitimidad de una fuente sin revelar su identidad. Proteccion criptografica de grado militar para denunciantes.

⭐

Reputacion de Fuentes

Sistema de scoring de reputacion basado en historial de verificacion. Cada fuente acumula puntuacion por precision, consistencia y transparencia. Decae con informacion falsa verificada.

👥

Fact-Check Comunitario

Verificacion impulsada por la comunidad con recompensas en $MATTR. Verificadores certificados validan informacion mediante consenso stake-weighted con disputas resueltas por DAO.

🤖

Anti-Deepfake

Deteccion por IA de imagenes y videos manipulados. Modelos de vision computacional entrenados en datasets soberanos detectan alteraciones con 99.2% de precision en tiempo real.

📚

Archivo Inmutable

Archivo blockchain de hechos verificados. Cada hecho confirmado se almacena permanentemente en MameyNode con hash criptografico, referencias cruzadas y timestamp inmutable.

🔗

API Publica

API abierta para navegadores y plataformas que consultan el estado de verificacion de cualquier contenido. Extension de navegador integrada, SDK para desarrolladores, webhook para editores.

🎓

Educacion Mediatica

Cursos de alfabetizacion mediatica integrados con el NEXUS Escolar. Pensamiento critico, identificacion de sesgos, verificacion de fuentes y resistencia a manipulacion informativa.

🚨

Alertas Ciudadanas

Alertas en tiempo real sobre campanas de desinformacion activas. Sistema de alerta temprana que detecta patrones de propagacion coordinada y notifica a ciudadanos y verificadores.

🔌 API Endpoints

REST + gRPC + WebSocket para integraciones
GET/api/v1/veritas/verify/{content_hash}

Estado de verificacion de contenido por hash. Retorna veracity score, fuentes y timestamp.

POST/api/v1/veritas/submit

Enviar contenido para verificacion comunitaria. Params: content, source_url, metadata.

GET/api/v1/veritas/source/{source_id}/reputation

Score de reputacion de una fuente con historial y tendencia.

POST/api/v1/veritas/whistleblower/submit

Envio anonimo con ZK-Proof. Params: zk_proof, encrypted_payload, commitment.

GET/api/v1/veritas/deepfake/analyze/{media_hash}

Analisis de deepfake para imagen o video. Retorna probabilidad de manipulacion.

GET/api/v1/veritas/alerts/active

Alertas activas de campanas de desinformacion con nivel de amenaza y cobertura.

GET/api/v1/veritas/archive/{fact_id}

Consultar hecho verificado en el archivo inmutable con pruebas y referencias.
