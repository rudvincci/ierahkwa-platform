## Usage Guidelines

### For Venture Capital
1. Start with **Investment One-Pager** for initial interest
2. Follow up with **Executive Investment Deck** for meetings
3. Provide **Investment Memorandum** for due diligence
4. Use **Financial Projections** for financial analysis
5. Review **Technology Due Diligence** for technical evaluation

### For Strategic Investors
1. Share **Investment One-Pager** for initial discussions
2. Present **Executive Investment Deck** in meetings
3. Deep dive with **Investment Memorandum** for strategic evaluation
4. Review **Competitive Analysis** for market positioning
5. Use **Market Opportunity Analysis** for market validation

### For Private Equity
1. **Investment Memorandum** for comprehensive review
2. **Financial Projections** for financial analysis
3. **Risk Assessment** for risk evaluation
4. **Exit Strategy** for exit planning
5. **Valuation Analysis** for valuation discussions

## Export Formats

### Recommended Formats
- **PDF**: For sharing and printing
- **PowerPoint**: For presentations (Executive Deck)
- **Excel**: For financial models (Financial Projections)
- **Markdown**: For version control and editing

### Conversion Tools
- **Markdown to PDF**: Pandoc, Marked, or online converters
- **Markdown to PowerPoint**: Pandoc or manual conversion
- **Financial Models**: Excel or Google Sheets

## Distribution Guidelines

### Confidentiality
- **Investment One-Pager**: NDA-required
- **Executive Investment Deck**: NDA-required, serious investors only
- **Investment Memorandum**: Highly confidential, serious investors only
- **Financial Projections**: Highly confidential, due diligence only
- **All Documents**: Confidential, authorized investors only

### Sharing Channels
- **Email**: Attach PDFs (password-protected recommended)
- **Secure Portal**: Use secure data rooms for due diligence
- **Presentations**: Use PowerPoint or PDF
- **Financial Models**: Use Excel or secure sharing
