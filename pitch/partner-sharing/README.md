## Usage Guidelines

### For Investors
1. Start with **One-Pager** for initial interest
2. Follow up with **Executive Summary Deck** for meetings

4. Use **Infographic** for visual presentations

### For Partners
1. Share **One-Pager** for initial discussions
2. Present **Executive Summary Deck** in meetings
3. Deep dive with **Architecture Overview** for integration planning
4. Use **Infographic** for marketing and sales materials

1. **One-Pager** for quick updates
2. **Executive Summary Deck** for board meetings
3. **Infographic** for public-facing materials

### Recommended Formats
- **PDF**: For sharing and printing
- **PowerPoint**: For presentations (Executive Deck)

- **HTML**: For web embedding
- **Markdown**: For version control and editing

- **Markdown to PowerPoint**: Pandoc or manual conversion
