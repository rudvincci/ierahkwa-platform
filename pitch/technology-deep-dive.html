Post-Quantum Security

NIST-standardized algorithms — quantum-resistant by design

SIGNATURES ML-DSA-65

Module Lattice Digital Signature Algorithm (FIPS 204). Replaces RSA/ECDSA for all sovereign document signing, identity attestation, and chain transactions.

ENCRYPTION ML-KEM-1024

Module Lattice Key Encapsulation Mechanism (FIPS 203). Secures all TLS connections, data-at-rest encryption, and inter-service communication.

IDENTITY Zero-Knowledge Proofs

Prove identity attributes without revealing underlying data. Age verification, citizenship, and credential proofs without data exposure.

NETWORK Isolation

All services bind to 127.0.0.1. Docker sovereign-net (172.28.0.0/16) provides network-level isolation. Zero external attack surface.
