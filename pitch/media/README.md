## Export Formats

### Recommended Formats
- **PDF**: For sharing and printing
- **Word**: For editing and customization
- **Markdown**: For version control and editing
- **HTML**: For web embedding

### Conversion Tools
- **Markdown to PDF**: Pandoc, Marked, or online converters
- **Markdown to Word**: Pandoc or manual conversion
- **Images**: High-resolution formats (PNG, JPG, SVG)
