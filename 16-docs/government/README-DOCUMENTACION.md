## 4. Resumen: qué recibes con el software

- **Archivos** necesarios para ejecutar la plataforma (según EULA).  
- **Autorización legal** de uso (EULA, Certificado de Licencia).  
- **Información de uso**: Manual de Usuario.  
- **Información de instalación y configuración**: Manual de Instalación/Configuración.  
- **Información técnica**: Documentación Técnica, Librerías y Componentes.  
- **Acceso**: portal, soporte, actualizaciones (según tipo de licencia).  
- **Derechos**: uso personal, comercial o gubernamental; número de usuarios según contrato.  
- **Clave/Serial/Token**: para activar y validar la licencia (si aplica).
