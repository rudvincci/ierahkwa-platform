## 🚀 Quick Start

```bash
# Clonar
git clone https://github.com/red-soberana/red-soberana.git
cd red-soberana

# Ver portal central
open portal-soberano.html

# Ver cualquier plataforma
open plataformas/01-CorreoSoberano/index.html

# Deploy con Docker
cd infra/docker && docker-compose up -d

# Deploy en K8s
kubectl apply -f infra/k8s/
```

## ⛓ Stack Técnico

- **Blockchain:** MameyNode v4.2 (EVM-compatible, 12,847 TPS)
- **Moneda:** Wampum (Ⓦ) via BDET Bank
- **AI:** 42 motores soberanos (AI Fortress)
- **Cifrado:** Post-quantum E2E (Kyber/Dilithium)
- **Telecom:** Satelital + Mesh 5G
- **Idiomas:** 14 indígenas + 6 globales (Atabey Translator)
- **Frontend:** HTML5/CSS3/JS (cada plataforma es self-contained)
- **Backend:** Node.js + PostgreSQL + MameyNode
- **Infra:** Docker + Kubernetes + CloudSoberana
