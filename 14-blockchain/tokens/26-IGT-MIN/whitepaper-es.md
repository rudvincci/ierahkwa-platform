# IGT-MIN: Token de Ministerio de Industria
## Whitepaper v1.0
### Gobierno Soberano de Ierahkwa Ne Kanienke

---

## Resumen Ejecutivo

El **IGT-MIN (Token de Ministerio de Industria)** es un token gubernamental oficial del Gobierno Soberano de Ierahkwa Ne Kanienke. Este token representa desarrollo industrial, manufactura e iniciativas de crecimiento económico.

---

## Especificaciones del Token

| Atributo | Valor |
|----------|-------|
| **Nombre** | Ministerio de Industria |
| **Símbolo** | IGT-MIN |
| **Suministro Total** | 10,000,000,000,000 (10 Trillones) |
| **Decimales** | 9 |
| **Estándar del Token** | Ierahkwa Government Token (IGT) |
| **Blockchain** | Ierahkwa Sovereign Blockchain (ISB) |
| **ID de Departamento** | 26 |
| **Categoría** | Resources & Development |
| **Nivel de Autoridad** | Ministerial |

## Infraestructura Blockchain

- **Blockchain**: Ierahkwa Sovereign Blockchain (ISB)
- **Nodo**: Ierahkwa Futurehead Mamey Node
