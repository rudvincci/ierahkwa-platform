# IGT-MST: Token de Ministerio de Ciencia y Tecnología
## Whitepaper v1.0
### Gobierno Soberano de Ierahkwa Ne Kanienke

---

## Resumen Ejecutivo

El **IGT-MST (Token de Ministerio de Ciencia y Tecnología)** es un token gubernamental oficial del Gobierno Soberano de Ierahkwa Ne Kanienke. Este token representa investigación científica, innovación tecnológica y avance digital.

---

## Especificaciones del Token

| Atributo | Valor |
|----------|-------|
| **Nombre** | Ministerio de Ciencia y Tecnología |
| **Símbolo** | IGT-MST |
| **Suministro Total** | 10,000,000,000,000 (10 Trillones) |
| **Decimales** | 9 |
| **Estándar del Token** | Ierahkwa Government Token (IGT) |
| **Blockchain** | Ierahkwa Sovereign Blockchain (ISB) |
| **ID de Departamento** | 29 |
| **Categoría** | Resources & Development |
| **Nivel de Autoridad** | Ministerial |
