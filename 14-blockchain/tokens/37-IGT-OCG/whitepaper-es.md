## Resumen Ejecutivo

El **IGT-OCG (Token de Oficina del Contralor General)** es un token gubernamental oficial del Gobierno Soberano de Ierahkwa Ne Kanienke. Este token representa supervisión financiera, auditoría y responsabilidad fiscal.

---

| Atributo | Valor |

| **Nombre** | Oficina del Contralor General |
| **Símbolo** | IGT-OCG |
| **Suministro Total** | 10,000,000,000,000 (10 Trillones) |
| **Decimales** | 9 |
| **Estándar del Token** | Ierahkwa Government Token (IGT) |
| **Blockchain** | Ierahkwa Sovereign Blockchain (ISB) |
