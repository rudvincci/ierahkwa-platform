# IGT-OO: Token de Oficina del Defensor del Pueblo
## Whitepaper v1.0
### Gobierno Soberano de Ierahkwa Ne Kanienke

---

## Resumen Ejecutivo

El **IGT-OO (Token de Oficina del Defensor del Pueblo)** es un token gubernamental oficial del Gobierno Soberano de Ierahkwa Ne Kanienke. Este token representa defensa del ciudadano, responsabilidad gubernamental y resolución de quejas.

---

## Especificaciones del Token

| Atributo | Valor |
|----------|-------|
| **Nombre** | Oficina del Defensor del Pueblo |
| **Símbolo** | IGT-OO |
| **Suministro Total** | 10,000,000,000,000 (10 Trillones) |
| **Decimales** | 9 |
| **Estándar del Token** | Ierahkwa Government Token (IGT) |
| **Blockchain** | Ierahkwa Sovereign Blockchain (ISB) |
| **ID de Departamento** | 38 |
| **Categoría** | Security & Administration |
| **Nivel de Autoridad** | Oversight |
