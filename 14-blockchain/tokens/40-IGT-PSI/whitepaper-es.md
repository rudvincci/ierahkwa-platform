# IGT-PSI: Token de Servicio Postal de Ierahkwa
## Whitepaper v1.0
### Gobierno Soberano de Ierahkwa Ne Kanienke

---

## Resumen Ejecutivo

El **IGT-PSI (Token de Servicio Postal de Ierahkwa)** es un token gubernamental oficial del Gobierno Soberano de Ierahkwa Ne Kanienke. Este token representa entrega de correo, servicios postales y logística de comunicación.

---

## Especificaciones del Token

| Atributo | Valor |
|----------|-------|
| **Nombre** | Servicio Postal de Ierahkwa |
| **Símbolo** | IGT-PSI |
| **Suministro Total** | 10,000,000,000,000 (10 Trillones) |
| **Decimales** | 9 |
| **Estándar del Token** | Ierahkwa Government Token (IGT) |
| **Blockchain** | Ierahkwa Sovereign Blockchain (ISB) |
| **ID de Departamento** | 40 |
| **Categoría** | Security & Administration |
| **Nivel de Autoridad** | Administrative |

## Infraestructura Blockchain

- **Blockchain**: Ierahkwa Sovereign Blockchain (ISB)
- **Nodo**: Ierahkwa Futurehead Mamey Node
- **Banco Central**: Ierahkwa Futurehead BDET Bank
- **Consenso**: Prueba de Autoridad Soberana (SPoA)
