# IGT-CRO: Token de Oficina del Registro Civil
## Whitepaper v1.0
### Gobierno Soberano de Ierahkwa Ne Kanienke

---

## Resumen Ejecutivo

El **IGT-CRO (Token de Oficina del Registro Civil)** es un token gubernamental oficial del Gobierno Soberano de Ierahkwa Ne Kanienke. Este token representa documentación civil, registros vitales y servicios de identificación ciudadana.

---

## Especificaciones del Token

| Atributo | Valor |
|----------|-------|
| **Nombre** | Oficina del Registro Civil |
| **Símbolo** | IGT-CRO |
| **Suministro Total** | 10,000,000,000,000 (10 Trillones) |
| **Decimales** | 9 |
| **Estándar del Token** | Ierahkwa Government Token (IGT) |
| **Blockchain** | Ierahkwa Sovereign Blockchain (ISB) |
| **ID de Departamento** | 35 |
| **Categoría** | Security & Administration |
| **Nivel de Autoridad** | Administrative |
