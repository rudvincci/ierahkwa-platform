# IGT-MCT: Token de Ministerio de Comercio
## Whitepaper v1.0
### Gobierno Soberano de Ierahkwa Ne Kanienke

---

## Resumen Ejecutivo

El **IGT-MCT (Token de Ministerio de Comercio)** es un token gubernamental oficial del Gobierno Soberano de Ierahkwa Ne Kanienke. Este token representa regulación comercial, comercio internacional y desarrollo empresarial.

---

## Especificaciones del Token

| Atributo | Valor |
|----------|-------|
| **Nombre** | Ministerio de Comercio |
| **Símbolo** | IGT-MCT |
| **Suministro Total** | 10,000,000,000,000 (10 Trillones) |
| **Decimales** | 9 |
| **Estándar del Token** | Ierahkwa Government Token (IGT) |
| **Blockchain** | Ierahkwa Sovereign Blockchain (ISB) |
| **ID de Departamento** | 25 |
| **Categoría** | Resources & Development |
| **Nivel de Autoridad** | Ministerial |

---

## Infraestructura Blockchain

- **Blockchain**: Ierahkwa Sovereign Blockchain (ISB)
- **Nodo**: Ierahkwa Futurehead Mamey Node
- **Banco Central**: Ierahkwa Futurehead BDET Bank
- **Consenso**: Prueba de Autoridad Soberana (SPoA)
