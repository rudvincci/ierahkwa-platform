# IGT-PM: Token de Oficina del Primer Ministro
## Whitepaper v1.0
### Gobierno Soberano de Ierahkwa Ne Kanienke

---

## Resumen Ejecutivo

El **IGT-PM (Token de Oficina del Primer Ministro)** es un token gubernamental oficial del Gobierno Soberano de Ierahkwa Ne Kanienke. Este token representa autoridad ejecutiva y liderazgo.

---

## Especificaciones del Token

| Atributo | Valor |
|----------|-------|
| **Nombre** | Oficina del Primer Ministro |
| **Símbolo** | IGT-PM |
| **Suministro Total** | 10,000,000,000,000 (10 Trillones) |
| **Decimales** | 9 |
| **Estándar del Token** | Ierahkwa Government Token (IGT) |
| **Blockchain** | Ierahkwa Sovereign Blockchain (ISB) |
| **ID de Departamento** | 01 |
| **Categoría** | Executive & Core |
| **Nivel de Autoridad** | Executive |
