# IGT-MFT: Token de Ministerio de Finanzas y Tesoro
## Whitepaper v1.0
### Gobierno Soberano de Ierahkwa Ne Kanienke

---

## Resumen Ejecutivo

El **IGT-MFT (Token de Ministerio de Finanzas y Tesoro)** es un token gubernamental oficial del Gobierno Soberano de Ierahkwa Ne Kanienke. Este token representa política fiscal, gestión presupuestaria y gobernanza económica.

---

## Especificaciones del Token

| Atributo | Valor |
|----------|-------|
| **Nombre** | Ministerio de Finanzas y Tesoro |
| **Símbolo** | IGT-MFT |
| **Suministro Total** | 10,000,000,000,000 (10 Trillones) |
| **Decimales** | 9 |
| **Estándar del Token** | Ierahkwa Government Token (IGT) |
| **Blockchain** | Ierahkwa Sovereign Blockchain (ISB) |
| **ID de Departamento** | 03 |
| **Categoría** | Executive & Core |
| **Nivel de Autoridad** | Ministerial |
