# IGT-NPS: Token de Servicio Nacional de Policía
## Whitepaper v1.0
### Gobierno Soberano de Ierahkwa Ne Kanienke

## Resumen Ejecutivo

El **IGT-NPS (Token de Servicio Nacional de Policía)** es un token gubernamental oficial del Gobierno Soberano de Ierahkwa Ne Kanienke. Este token representa aplicación de la ley, seguridad pública y protección comunitaria.

---

| Atributo | Valor |
|----------|-------|
| **Nombre** | Servicio Nacional de Policía |
| **Símbolo** | IGT-NPS |
| **Suministro Total** | 10,000,000,000,000 (10 Trillones) |
| **Decimales** | 9 |
| **Estándar del Token** | Ierahkwa Government Token (IGT) |
| **Blockchain** | Ierahkwa Sovereign Blockchain (ISB) |
| **ID de Departamento** | 31 |
| **Categoría** | Security & Administration |
