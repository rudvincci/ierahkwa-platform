# IGT-MJ: Token de Ministerio de Justicia
## Whitepaper v1.0
### Gobierno Soberano de Ierahkwa Ne Kanienke

---

## Resumen Ejecutivo

El **IGT-MJ (Token de Ministerio de Justicia)** es un token gubernamental oficial del Gobierno Soberano de Ierahkwa Ne Kanienke. Este token representa administración de justicia, asuntos legales y estado de derecho.

---

## Especificaciones del Token

| Atributo | Valor |
|----------|-------|
| **Nombre** | Ministerio de Justicia |
| **Símbolo** | IGT-MJ |
| **Suministro Total** | 10,000,000,000,000 (10 Trillones) |
| **Decimales** | 9 |
| **Estándar del Token** | Ierahkwa Government Token (IGT) |
| **Blockchain** | Ierahkwa Sovereign Blockchain (ISB) |
| **ID de Departamento** | 04 |
| **Categoría** | Executive & Core |
| **Nivel de Autoridad** | Ministerial |

---

## Infraestructura Blockchain

- **Blockchain**: Ierahkwa Sovereign Blockchain (ISB)
- **Nodo**: Ierahkwa Futurehead Mamey Node
- **Banco Central**: Ierahkwa Futurehead BDET Bank
- **Consenso**: Prueba de Autoridad Soberana (SPoA)
